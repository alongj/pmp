# 5.1 项目范围

1.  范围包括
   1. 产品范围——最终交付的成果
   2. 项目范围——为了实现成果实现的管理工作
2. 过程
   1. 确保 
3. 不同生命周期的范围管理对比

|                 | 预测型         | 迭代/增量型  | 适应型   |
| --------------- | -------------- | ------------ | -------- |
| 定义范围的时间  | 项目开始       | 每个迭代开始 | 随时     |
| 确认范围的时间  | 项目或阶段结束 | 每个迭代结束 | 随时反馈 |
| 范围控制文件    | 范围基准       | 版本配置文件 | 用户故事 |
| 发起人/客户参与 | 里程碑         | 周期性       | 持续性   |
